# Labs

## 目的 / Purpose

實驗、demo、prototype、research 的空間。

This directory is for experiments, demos, prototypes, and research.

## 特徵 / Characteristics

- 不一定可發佈 / Not necessarily publishable
- 可以較自由 / More freedom allowed
- 重點是探索與驗證 / Focus on exploration and validation

## 內容範例 / Content Examples

- Prototype applications
- Demo apps
- RAG 實驗 / RAG experiments
- 失敗案例 / Failed attempts (learning experiences)
- 創意來源 / Creative ideas
- Architecture exploration

## 演化路徑 / Evolution Path

```
labs/idea
  -> patterns/prototype
  -> tools/usable tool
  -> skills/reusable knowledge
  -> playbooks/documented execution
```

實驗成功後可提煉為 patterns、tools 或 skills。

Successful experiments can be distilled into patterns, tools, or skills.
