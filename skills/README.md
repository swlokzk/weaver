# Skills

## 目的 / Purpose

可直接安裝或複用的 skill packs。

Directly installable or reusable skill packs.

## 特徵 / Characteristics

- 不是心得文 / Not just reflections or notes
- 應有明確用途 / Should have clear purposes
- 有使用時機與結果導向 / Timing and result-oriented

## 內容範例 / Content Examples

- Agent skill definitions
- Reusable execution knowledge
- Skill packs that can be installed

## 理解方式 / Understanding

Skills 是從 labs 和 tools 提煉出來的可重用知識資產。

Skills are reusable knowledge assets distilled from labs and tools.

## MCP-native Skills

Skill 不只是 markdown，還能：
- 被 tools 使用 / Used by tools
- 被 runtime 組合 / Composed by runtime
- 被 CI 檢查 / Checked by CI
- 被 agent 調用 / Called by agents
