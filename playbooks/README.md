# Playbooks

## 目的 / Purpose

可人工閱讀與執行的方法手冊。

Human-readable and executable method manuals.

## 特徵 / Characteristics

- 偏流程 / Process-oriented
- 偏操作方法 / Operation method-focused
- 偏可落地執行 / Execution-ready

## 內容範例 / Content Examples

- 作戰手冊 / Operation manuals
- Step-by-step methods
- Agent/human hybrid workflows

## 使用場景 / Usage Scenarios

Playbooks 適合需要明確步驟、可重複執行的工作流程。

Playbooks are suitable for workflows that require clear steps and repeatable execution.
