# Patterns

## 目的 / Purpose

可重用模式、結構設計、解法框架。

Reusable patterns, structural designs, and solution frameworks.

## 特徵 / Characteristics

- 比 templates 更不易混淆 / Less ambiguous than templates
- 適合 agentic learning / Suitable for agentic learning
- 強調可抽象、可套用 / Emphasizes abstraction and applicability

## 內容範例 / Content Examples

- 可重用模式 / Reusable patterns
- 結構設計 / Structural designs
- 解法框架 / Solution frameworks
- Repo-to-skill methods
- Lab-to-skill methods

## 理解方式 / Understanding

Patterns 是從實驗和實作中提煉出來的抽象方法，可以套用到不同場景。

Patterns are abstract methods distilled from experiments and implementations that can be applied to different scenarios.
